# 🎬 YouTube Automation Agent

Fully automated YouTube content management system with AI-powered content generation, analytics, and multi-channel management.

## Features
- Dashboard with real-time stats
- Multi-channel management
- AI content generation pipeline
- Analytics & SEO optimization
- One-click deploy to Render.com

## Deploy
Click the button below to deploy to Render.com (FREE):

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

1. Connect your GitHub account
2. Select this repository
3. Deploy!

## Tech Stack
- React + TypeScript + Tailwind CSS
- Hono.js (Node.js server)
- SQLite + Prisma ORM
- Recharts (analytics charts)
